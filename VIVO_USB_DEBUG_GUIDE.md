# 📱 vivo X Fold5 开启开发者模式和 USB 调试指南

## 🔧 开启开发者模式

### 步骤 1: 进入设置
1. 打开手机**设置**
2. 向下滚动找到**关于手机**或**更多设置 → 关于手机**

### 步骤 2: 连续点击版本号
1. 在"关于手机"页面找到**软件版本号**或**版本号**
2. **连续快速点击 7 次**版本号
3. 会提示"您已处于开发者模式"或"开发者模式已开启"

## 🔌 开启 USB 调试

### 步骤 3: 进入开发者选项
1. 返回**设置**主页面
2. 找到**更多设置 → 开发者选项**
   - 或者直接在设置搜索栏搜索"开发者选项"

### 步骤 4: 开启 USB 调试
1. 在开发者选项页面，找到并开启：
   - ✅ **USB 调试**
   - ✅ **USB 安装**（允许通过 USB 安装应用）
   - ✅ **USB 调试（安全设置）**（如果有的话）

2. 会弹出警告对话框，点击**确定**

### 步骤 5: 连接电脑
1. 使用 USB 数据线连接手机和电脑
2. 手机屏幕会弹出**允许 USB 调试**的对话框
3. **勾选"始终允许使用这台计算机进行调试"**
4. 点击**确定**或**允许**

## 🧪 验证连接

完成以上步骤后，在电脑终端运行：
```bash
adb devices
```

应该会看到类似输出：
```
List of devices attached
12345678        device
```

如果显示 `device`，说明连接成功！

## ⚠️ 常见问题

### 问题 1: 没有"开发者选项"
**解决:** 重复步骤 2，确保连续点击版本号 7 次

### 问题 2: USB 调试灰色无法开启
**解决:** 
- 检查是否有企业管理限制
- 检查是否有家长控制
- 尝试重启手机

### 问题 3: 手机没有弹出授权对话框
**解决:**
1. 重新插拔 USB 线
2. 在开发者选项中关闭再开启 USB 调试
3. 更换 USB 接口
4. 确认使用数据线而非充电线

### 问题 4: 连接显示 unauthorized
**解决:**
1. 在手机上点击允许
2. 运行 `adb kill-server && adb start-server`
3. 再次运行 `adb devices`

## 📋 vivo 特定注意事项

vivo 手机可能还需要：
1. **开启"允许模拟点击"**（在开发者选项中）
2. **关闭"纯净模式"**（设置 → 安全与隐私 → 纯净模式）
3. **允许安装未知应用**（设置 → 安全与隐私 → 更多安全设置 → 安装未知应用）
