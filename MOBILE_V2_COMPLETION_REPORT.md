## 🎨 设计系统

### 颜色方案
```
主色调: 渐变紫（#667eea → #764ba2）
背景色: #f5f7fa
卡片色: #ffffff
文字色: #333 / #666 / #999
成功色: #2ed573 / #2a8a4e
警告色: #ffa502 / #856404
危险色: #ff4757 / #c33
```

### 视觉元素
- **圆角**: 8px - 20px（卡片级别不同）

- **点击**: scale(0.98) 缩放反馈
- **悬停**: 阴影加深

## 🔧 技术实现

### 路由系统
```typescript
/login          → LoginView
/servers        → ServerSelectView
/instances      → InstanceListView
/tasks          → TasksView
/tasks/:id      → TaskDetailView
/settings       → SettingsView
```

### 状态管理
```typescript
// LocalStorage
- pocket_user: 用户信息
- selected_server: 当前服务器
- selected_instance: 当前实例
```

### 实时更新
```typescript
// WebSocket 事件
- task_created: 任务创建
- task_updated: 任务更新
- session_attached: 会话附加
```

## 🎯 用户使用流程

### 首次使用
1. **安装 APK** - 从下载文件夹安装
2. **打开应用** - 看到登录页面
3. **登录** - 输入 admin/admin
4. **选择服务器** - 选择 NPS 56 或 252
5. **选择实例** - 选择要管理的 OpenCode 实例
6. **查看任务** - 看到按状态分组的任务列表
7. **创建任务** - 点击 + 号创建新任务
8. **查看详情** - 点击任务查看详细信息
9. **附加会话** - 将 OpenCode 会话关联到任务

### 日常使用
1. 打开应用 → 自动登录（已保存）
2. 进入上次使用的实例
3. 查看任务列表
4. 创建/管理任务
5. 附加会话
6. 底部导航快速切换
